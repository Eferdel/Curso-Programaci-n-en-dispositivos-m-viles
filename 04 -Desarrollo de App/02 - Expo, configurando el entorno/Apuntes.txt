Expo: poder crear app hibridas con REACT y luego publicarlas sin robleas en ambas plataformas

https://expo.dev/tools

- iniciar un proyecto cualquira
npm install --global expo-cli

- Instalar modulos para el proyecto
expo init my-app
	- minimo
	- tabs con typescript (con ya pestañitas)
	- blank con typescript
	- blank *
* expo init my-app --npm (para no tener que instalar yarn)

- Iniciar la aplicación
expo start

Para poder ejecutar nuestra app directamente en nuestro dipositivo usamos expo go
